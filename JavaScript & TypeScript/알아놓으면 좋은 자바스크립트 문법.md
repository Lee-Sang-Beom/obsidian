### Shorthand property names

- JavaScript의 Object는 항상 Key와 value로 구성되어 있습니다.
- 만약 value에 변수명을 사용할 예정인데, key와 이름이 같은 경우 축약하여, 아래와 같이 작성할 수 있습니다.

```jsx
// 일반적인 경우
const obj1 = {
	name: 'Lee',
	age: 25,
};

const name = "Lee";
const age = 25;

// key, value명이 동일한 경우 축약하여 사용가능
const obj2 = {
	name, // name: name,
	age, // age: age,

};
```
