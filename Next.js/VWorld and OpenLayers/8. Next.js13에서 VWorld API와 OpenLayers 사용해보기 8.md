#### 2. zoomLevel 관리하기

- 먼저, zoomLevel을 관리할 수 있는 state를 정의하자.
	- 이제 이 `zoomLevel` state에 따라 지도에 어떤 Feature가 올라갈 지 결정될 것이다.
```tsx
const [zoomLevel, setZoomLevel] = useState<number>(13.8);
```
