## 1. 서비스 개요

AI 기반 쇼츠 영상 자동 제작 플랫폼. 분야 선택부터 다운로드까지 9단계 워크플로우로 구성된다.

| 제작 메뉴      | 상태               |
| ---------- | ---------------- |
| 일반 제작      | ✅ 테스트 완료         |
| 퀴즈 제작      | ✅ 테스트 완료         |
| 캐릭터 스토리 제작 | ✅ 테스트 완료         |
| 쇼호스트 제작    | ⚠️ 작업 중 (테스트 제외) |

## 2. 영상제작(3개 영상제작 공통) 단계별 테스트 결과

| 단계        | 기능 요약                                   | 결과        |
| --------- | --------------------------------------- | --------- |
| ① 분야 선택   | 20개+ 카테고리, 직접 추가 가능                     | ✅         |
| ② 트렌드 분석  | YouTube·웹 기반 트렌드 자동 분석, 약 30초 소요        | ⚠️ 간헐적 실패 |
| ③ 주제 추천   | 트렌드 점수·경쟁도·예상 조회수 포함 AI 추천 5개 + 직접 입력   | ✅         |
| ④ 스크립트 생성 | 나레이션 + 이미지/비디오 프롬프트 자동 생성 (10컷)         | ✅         |
| ⑤ 스크립트 편집 | 컷별 나레이션·프롬프트 편집, 이미지/영상 직접 업로드로 컷 대체 가능 | ⚠️ 버그 있음  |
| ⑥ 콘텐츠 생성  | TTS → 이미지 생성 → 영상 합성, 약 30~60초 소요       | ❌ 오류 빈번   |
| ⑦ 콘텐츠 검수  | 컷별/완성본 미리보기, 메타데이터·BGM 설정, 재생성          | ⚠️ 버그 있음  |
| ⑧~⑨ 다운로드  | 완성 영상 다운로드                              | ❌ 오류 빈번   |
