## 4. 매물 등록 (판매)

### TC-007 — 매물 등록 및 판매 플로우

|#|검증 항목|결과|
|---|---|:-:|
|**판매 과정**|||
|1|`/sell` 페이지 접근 가능한가 (비로그인 시 로그인 페이지로 이동하는가)|✅|
|2|서비스명, 설명, 매물 로고 이미지, 카테고리 등 단계별 기본 정보 입력이 되는가|✅|
|3|이전/다음에 따라 입력정보가 유지되고 있는가|✅|
|4|등록 완료 후 내 매물 목록에 표시되는가|✅|
|5|등록된 매물이 수정 가능하며, 수정 오류가 없는가|✅|
|**구매자 제안 시**|||
|6|검토 중일 때, 구매자가 해당 매물을 구매할 수 없도록 처리했는가|❌|
|7|상대방과 내 채팅이 잘 보이는가|✅|
|8|단계 진행에 따라 거래 단계의 진행상태가 변경되는가|⚠️|
|9|자료 열람 시, 데이터룸이 양측 전부 활성화되고 파일 업로드가 이루어지는가|❌|
|10|판매자가 최종 협상 가격을 정상적으로 변경 가능한가|✅|
|11|거래 완료 시, 거래 현황 페이지를 나갈 수 있는 별도 이벤트 트리거가 있는가|❌|

- **판정:** ❌
- **비고**
    - **#6:** 검토중인 매물 상세페이지 링크 접속 시, 검토중임에도 거래가 시작되어버림
    - **#8:** 단계 진행 시 반대측 사용자는 항상 수동으로 새로고침을 진행해야 함 (자동 갱신 미지원)
    - **#8:** 거래 종료 시 거래단계 중 6번 항목이 활성화되지 않음
    - **#9:** 파일 업로드 불가 — 오류 메시지: `Bucket not found`
    - **#11:** #6 상황 발생 상태에서 거래 완료 후 페이지를 정상적으로 이탈할 수 있는 트리거 버튼이 존재하지 않음
- **추가 테스트 필요**
    - 정상 매물 등록 후 거래 플로우 전체 체크 필요
    - 정상 매물 판매 완료 시, 거래 현황 페이지 이탈 트리거 존재 여부 검토 필요
