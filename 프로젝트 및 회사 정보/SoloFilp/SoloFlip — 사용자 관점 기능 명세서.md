**접근 권한:** 누구나

---

### 기능 목록

|#|기능|설명|
|---|---|---|
|1|서비스 소개|히어로 섹션에서 서비스 목적과 핵심 기능(Stripe 매출 검증, AI 가격 분석, 에스크로 거래)을 노출한다|

## 2. 회원가입

**경로:** `/signup`  
**접근 권한:** 비로그인 사용자  
**목적:** 신규 계정 생성

---

### 기능 목록

|#|기능|설명|
|---|---|---|
|1|이메일 입력|사용자가 이메일 주소를 입력한다|
|2|비밀번호 입력|사용자가 비밀번호를 입력한다|
|3|유효성 검사|이메일 형식, 비밀번호 길이, 필수 항목 미입력 시 오류 메시지를 노출한다|
|4|중복 이메일 검사|이미 가입된 이메일로 재가입 시도 시 안내 메시지를 노출한다|
|5|가입 처리|입력 정보를 서버에 전송하고, 가입 완료 시 로그인 페이지로 이동한다|
|6|로딩 상태 표시|가입 요청 중 로딩 UI를 노출한다|

---

### 사용자 흐름

#### [흐름 1] 정상 가입

```
/signup 접속
  └─► 이메일, 비밀번호 입력
        └─► [가입하기] 클릭
              └─► 로딩 UI 표시
                    └─► 가입 완료 → 안내 메시지 노출 → /login 으로 이동
```

#### [흐름 2] 유효성 오류

```
/signup 접속
  └─► 이메일 형식 오류 또는 비밀번호 짧음 또는 필수 항목 미입력
        └─► [가입하기] 클릭
              └─► 인라인 오류 메시지 노출 (페이지 이동 없음)
```

#### [흐름 3] 중복 이메일

```
/signup 접속
  └─► 이미 가입된 이메일 입력
        └─► [가입하기] 클릭
              └─► "이미 사용 중인 이메일입니다" 안내 메시지 노출
```

---

### 유효성 검사 규칙

|필드|조건|오류 메시지|
|---|---|---|
|이메일|이메일 형식이 아닌 경우|올바른 이메일 형식을 입력해주세요|
|이메일|이미 가입된 이메일인 경우|이미 사용 중인 이메일입니다|
|비밀번호|최소 길이 미충족 시|비밀번호는 최소 N자 이상이어야 합니다|
|전체|필수 항목 미입력 시|필수 항목을 입력해주세요|


## 5. 매물 상세

**경로:** `/listing/:id`  
**접근 권한:** 누구나 (일부 기능은 로그인 필요)  
**목적:** 특정 매물의 상세 정보를 확인하고, 인수 제안 또는 관심 저장을 할 수 있다

---

### 기능 목록

|#|기능|설명|로그인 필요|
|---|---|---|:-:|
|1|서비스 정보 확인|서비스명, 설명, 카테고리, 월 매출, 성장률, 희망 판매가 등을 확인한다|✗|
|2|관심 매물 저장/해제|북마크 버튼으로 관심 매물에 저장하거나 해제한다|✓|
|3|인수 제안|[인수 제안] 버튼 클릭 시 거래 프로세스가 시작된다|✓|
|4|인수 진행 단계 확인|이미 인수 진행 중인 경우, 현재 거래 단계를 확인할 수 있다|✓|
|5|목록으로 복귀|뒤로가기 또는 네비게이션으로 `/browse` 로 돌아간다|✗|

---

### 사용자 흐름

#### [흐름 1] 매물 정보 확인 후 인수 제안

```
/listing/:id 접속
  └─► 서비스 정보 확인
        └─► [인수 제안] 버튼 클릭
              ├─► 비로그인 → /login 으로 이동
              └─► 로그인 → 거래 프로세스 시작 (거래 페이지로 이동)
```

#### [흐름 2] 관심 매물 저장

```
/listing/:id 접속 (로그인 상태)
  └─► 북마크(관심 저장) 버튼 클릭
        └─► 관심 매물 저장 완료 → 마이페이지 > 관심 매물에 추가됨
```

#### [흐름 3] 인수 진행 중 단계 확인

```
/listing/:id 접속 (이미 인수 제안 한 경우)
  └─► 현재 거래 단계 UI 노출
        └─► [거래 현황 보기] 클릭 → 거래 프로세스 페이지로 이동
```

## 6. 매물 등록 · 판매

**경로:** `/sell`  
**접근 권한:** 로그인 필요 (비로그인 시 `/login` 리다이렉트)  
**목적:** 사용자가 보유한 서비스를 단계별로 등록하고 매물로 게시한다

---

### 기능 목록

|#|기능|설명|
|---|---|---|
|1|단계별 정보 입력|서비스명, 설명, 카테고리, 로고 이미지, 월 매출, 희망 판매가 등을 단계별로 입력한다|
|2|이전 / 다음 이동|단계 이동 시 이전에 입력한 정보가 유지된다|
|3|매물 등록 완료|최종 제출 시 매물이 등록되고 내 매물 목록에 표시된다|
|4|등록 매물 수정|이미 등록한 매물의 정보를 수정할 수 있다|

---

### 사용자 흐름

#### [흐름 1] 매물 신규 등록

```
/sell 접속 (로그인 상태)
  └─► STEP 1: 서비스 기본 정보 입력 (서비스명, 카테고리, 설명, 로고)
        └─► [다음] 클릭
              └─► STEP 2: 매출 및 가격 정보 입력 (월 매출, 희망가 등)
                    └─► [다음] 클릭
                          └─► STEP 3: 최종 검토 및 제출
                                └─► [등록 완료] 클릭 → 매물 게시 → 내 매물 목록에 표시
```

#### [흐름 2] 등록 매물 수정

```
마이페이지 > 내 매물 접속
  └─► 등록된 매물 선택
        └─► [수정] 버튼 클릭 → 수정 폼으로 이동
              └─► 정보 수정 후 [저장] → 수정 완료
```

---

### 단계별 입력 항목

|단계|입력 항목|
|---|---|
|STEP 1|서비스명, 카테고리, 서비스 설명, 로고 이미지|
|STEP 2|월 매출, 희망 판매가, 성장률 등 수익 정보|
|STEP 3|입력 정보 최종 확인 및 제출|

## 7. 거래 프로세스

**경로:** 거래 상세 페이지 (매물 상세에서 인수 제안 후 진입)  
**접근 권한:** 로그인 필요, 해당 거래의 구매자 또는 판매자만 접근 가능  
**목적:** 인수 제안부터 거래 완료까지 단계별 프로세스를 진행한다

---

### 기능 목록

|#|기능|설명|대상|
|---|---|---|---|
|1|거래 단계 확인|현재 거래가 어느 단계인지 시각적으로 확인한다|양측|
|2|채팅|구매자와 판매자가 실시간으로 메시지를 주고받는다|양측|
|3|데이터룸|판매자가 자료를 업로드하고, 구매자가 열람한다|양측|
|4|가격 협상|판매자가 최종 협상 가격을 설정한다|판매자|
|5|단계 진행|각 단계를 완료 처리하여 다음 단계로 넘어간다|양측|
|6|거래 완료|최종 거래 완료 처리 후 페이지를 이탈한다|양측|
|7|알림|구매 제안 시 판매자에게 알림이 발송된다|판매자|

---

### 거래 단계 흐름

```
[STEP 1] 인수 제안
  └─► 구매자: 매물 상세에서 [인수 제안] 클릭
        └─► 판매자: 알림 수신 → 내 매물 페이지에서 제안 확인

[STEP 2] 검토 및 채팅
  └─► 판매자 / 구매자: 채팅으로 초기 협의 진행

[STEP 3] 자료 열람 (데이터룸)
  └─► 판매자: 관련 자료 파일 업로드
        └─► 구매자: 업로드된 자료 열람

[STEP 4] 가격 협상
  └─► 판매자: 최종 협상 가격 입력 및 확정

[STEP 5] 거래 진행
  └─► 양측: 단계별 확인 및 진행 처리

[STEP 6] 거래 완료
  └─► 거래 완료 처리 → 거래 현황 페이지 이탈 트리거 노출
```

---

### 사용자 행동별 흐름

#### [흐름 1] 구매자 — 인수 제안 및 거래 진행

```
/listing/:id 에서 [인수 제안] 클릭
  └─► 거래 페이지 진입
        └─► 채팅으로 판매자와 협의
              └─► 데이터룸에서 자료 열람
                    └─► 최종 가격 확인
                          └─► 거래 완료 처리
```

#### [흐름 2] 판매자 — 제안 수신 및 거래 진행

```
알림 또는 내 매물 페이지에서 제안 확인
  └─► 거래 페이지 진입
        └─► 채팅으로 구매자와 협의
              └─► 데이터룸에 자료 업로드
                    └─► 최종 협상 가격 설정
                          └─► 거래 완료 처리
```

## 8. 마이페이지

**접근 권한:** 로그인 필요  
**목적:** 내 계정 정보 관리, 관심 매물 확인, 내가 등록한 매물 관리

---

### 8-1. 내 정보

**경로:** `/my/profile` (또는 마이페이지 내 탭)

|#|기능|설명|
|---|---|---|
|1|내 정보 확인|현재 등록된 계정 정보(이메일, 닉네임 등)를 확인한다|
|2|내 정보 수정|변경할 정보를 입력하고 저장한다. 저장 완료 시 피드백을 노출한다|
|3|회원 탈퇴|계정을 영구 삭제한다 (현재 미구현)|

#### 사용자 흐름

```
마이페이지 > 내 정보 접속
  └─► 수정할 항목 변경
        └─► [저장] 클릭
              └─► 저장 완료 Toast 알림 노출
```

---

### 8-2. 관심 매물

**경로:** `/my/wishlist` (또는 마이페이지 내 탭)

|#|기능|설명|
|---|---|---|
|1|관심 매물 목록 확인|저장한 매물 목록을 확인한다|
|2|매물 상세 이동|관심 매물 클릭 시 해당 매물 상세 페이지로 이동한다|
|3|관심 해제|관심 매물을 해제하면 목록에서 제거된다. 완료 시 피드백을 노출한다|

#### 사용자 흐름

```
마이페이지 > 관심 매물 접속
  └─► 저장된 매물 목록 확인
        ├─► 매물 클릭 → /listing/:id 로 이동
        └─► 관심 해제 클릭 → 목록에서 제거 + 완료 피드백 노출
```

---

### 8-3. 내 매물

**경로:** `/my/listings` (또는 마이페이지 내 탭)

|#|기능|설명|
|---|---|---|
|1|내 매물 목록 확인|등록한 매물과 현재 상태(판매중 / 검토중 / 판매완료)를 확인한다|
|2|매물 상세 이동|매물 클릭 시 해당 상세 페이지로 이동한다|
|3|매물 수정|등록된 매물 정보를 수정한다|
|4|조회수 확인|매물별 조회수를 확인한다|

#### 사용자 흐름

```
마이페이지 > 내 매물 접속
  └─► 내가 등록한 매물 목록 + 상태 확인
        ├─► 매물 클릭 → /listing/:id 로 이동
        └─► [수정] 클릭 → 수정 폼으로 이동 → 수정 후 저장
```

#### 매물 상태 정의

|상태|설명|
|---|---|
|판매중|구매자가 인수 제안 가능한 상태|
|검토중|인수 제안이 들어와 거래가 진행 중인 상태. 추가 제안 불가|
|판매완료|거래가 완료된 상태|
