

## 테스트 범위 및 참고 사항

- **제작 페이지**: 일반 제작, 퀴즈 제작, 캐릭터 스토리 제작 총 3가지 기능이 존재함.

  

---

  

## 1. 버그 리포트 (Bug Reports)

  

### 1.1 텍스트 인코딩 깨짐

- **위치**: 전체 페이지

- **증상**: 일부 텍스트에서 인코딩이 깨져 출력됨. (예: `이 컷은 ��접 ���로드한 이미지을 사용합니��`)

  

### 1.2 마이페이지 로그인 상태 인식 오류

- **위치**: 마이페이지

- **증상**: 이미 로그인이 완료된 상태임에도 불구하고, 화면에 "로그인이 필요합니다"라는 안내 메시지가 노출됨.

  

### 1.3 구독 플랜 갱신 오류 (무료 회원으로 지속 표시)

- **위치**: 마이페이지 > 요금제/플랜 정보 표시 영역

- **증상**: 사용자가 유료 요금제나 다른 구독 플랜으로 변경을 완료했음에도, 권한 등급이 여전히 '무료 회원'으로 표시됨.

  

### 1.4 환경설정 저장 오류

- **위치**: 제작 페이지 전반 > 환경설정(⚙️)

- **증상**: 일반, 퀴즈 제작에 대한 환경설정 저장 시 에러가 발생하며 정상적으로 저장되지 않음.

  

### 1.5 트렌드 분석 간헐적 실패

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작, 캐릭터 스토리 제작 > 트렌드 분석

- **증상**: 트렌드 분석이 간헐적으로 실패함.

  

### 1.6 영상 생성 방식 미선택 시 유효성 검사 누락

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 스크립트 편집

- **증상**: 영상 생성 방식을 선택하지 않은 상태에서도 다음 단계로 진행이 가능하며, 이로 인해 이후 단계에서 오류가 발생함.

  

### 1.7 나레이션 스크립트 재작성 AI 무한 호출

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 스크립트 편집

- **증상**: 나레이션 스크립트 재작성 AI 기능을 횟수 제한 없이 무한으로 호출할 수 있음.

  

### 1.8 콘텐츠 생성 서버 오류 및 크레딧 미환불

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 생성

- **증상**:

  - 서버 오류(HTTP 500)가 빈번하게 발생함.

  - 생성 실패 시 차감된 크레딧이 환불되지 않는 경우가 다수 발생함. (실측: 4444 → 4430 → 4416)

  - 오류 발생 시 별도 안내 없이 이전 단계로 강제 이동됨.

  

### 1.9 TTS 오디오 생성 및 재생성 불가

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수 > 컷별 검수

- **증상**: 초기 TTS(Text-to-Speech) 오디오 생성뿐만 아니라, 이후 'TTS 재생성' 기능을 시도할 때에도 정상적으로 동작하지 않음. (오류 발생 또는 무반응)

  

### 1.10 컷 5 에셋 재생성 실패

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수 > 컷별 검수

- **증상**: 컷 5의 에셋 재생성이 빈번하게 실패함.

  

### 1.11 이미지 생성 및 재생성 실패

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수 > 컷별 검수

- **증상**:

  - 최초 이미지 생성 기능이 정상적으로 동작하지 않음.

  - 프롬프트 수정 후 '이미지 재생성'을 시도해도 계속해서 생성에 실패함.

  - **(추가 문제)** 화면상으로는 '이미지 생성 실패'로 표시되지만, 완성본 영상 검수 탭에서 영상을 확인해 보면 이미지가 들어간 영상이 출력되는 경우가 있음.

  

### 1.12 영상 생성 관련 예외 처리 누락

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수

- **증상**:

  - 필수 소스인 '이미지'와 'TTS'가 준비되지 않은 상태임에도 '영상 생성' 버튼이 활성화되어 클릭 가능한 상태임.

  - 해당 상태에서 영상 생성 버튼을 클릭할 경우, 검은 화면에 텍스트만 존재하는 영상 파일은 다운로드되나 리소스 파일은 받아지지 않음.

  

### 1.13 전체 재생성 버튼 무반응

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수

- **증상**: '전체 재생성' 버튼 클릭 시 아무런 반응이 없음.

  

### 1.14 에셋 재생성 중 단계 이동 가능

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수

- **증상**: 에셋 재생성이 진행 중인 상태에서도 다음 단계로 이동이 가능함.

  

### 1.15 완성본 영상 다운로드 예외 처리 누락 및 버튼 중복 클릭

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 콘텐츠 검수 > 완성본 영상 검수

- **증상**:

  - 필수 소스인 '이미지'와 'TTS'가 준비되지 않은 상태임에도 '완성본 영상 다운로드' 버튼이 활성화되어 클릭 가능한 상태임. 해당 상태에서 다운로드 버튼을 클릭할 경우, 검은 화면에 텍스트만 존재하는 영상 파일과 리소스 파일이 다운로드됨.

  - **(버튼 중복 클릭 문제)** '완성본 영상 다운로드' 버튼 클릭 후 렌더링 도중 '영상 생성' 버튼 클릭이 가능하며, 두 버튼을 동시에 누를 경우 '영상 생성' 버튼에 표시되는 진행률(progress)이 혼합되어 정상적으로 표시되지 않음.

  

### 1.16 다운로드 상태 표시 오류

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 다운로드

- **증상**: 실패 메시지가 표시되지만 파일은 실제로 다운로드되는 경우가 빈번함. (상태 표시 오류)

  

### 1.17 완성 영상 TTS 음성 누락

- **위치**: 제작 페이지 > 일반 제작, 퀴즈 제작 > 다운로드

- **증상**: 다운로드된 완성 영상에 TTS 음성이 누락되어 있음. (캐릭터 스토리 제작은 TTS 정상 적용 확인됨)
