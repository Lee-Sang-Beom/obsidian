
## 1. 버그 리포트 (Bug)

**[B-01] 비회원으로 무료 체험**
- 위치: (로그인하지 않은 상태) 로그인 페이지 혹은 메인 화면에서 제작 버튼 클릭 시 
- 증상	
	1. 로그인하지 않은 상태에서, 제작 버튼 클릭 시 로그인 페이지로 리다이엑트됨
	2. 비회원으로 무료 체험 버튼을 누르면 다시 메인 페이지로 이동
	3. 다시 영상 제작을 위해 제작 버튼을 누르면 로그인 페이지 이동
	4. 1~3번에 해당하는 과정이 반복됨

**[B-02] 텍스트 인코딩 문제**
![[텍스트 인코딩 문제.png|658]]
- 위치: 일반 제작 > 콘텐츠 검수(7단계)
- 증상: 제작 과정에서 표시되는 UI 내에서, 텍스트 인코딩 문제가 다수 존재함.

**[B-03] 무한 미디어 로딩**
![[무한 미디어 로딩.png|230]]
- 위치: 일반 제작 > 콘텐츠 검수 (7단계)
- 증상
	- 콘텐츠 검수단계로 넘어온 후, 컷 미리보기에서 표시되는 미디어 로딩 중 상태가 무한히 발생하는 문제
	- 상기 이미지에서는 진행도 20%에서 진행도가 올라가지 않음 (대기시간: 약 20분)

**[B-04] 다운로드 실패 UI 출력. 하지만 다운로드는 진행**
![[다운로드 실패라고 뜨지만, 다운로드는 됨.png|506]]
- 위치: 일반 제작 > 다운로드 완료 (9단계)
- 증상
	- 최종 결과에서 '다운로드 대상이 설정되지 않았습니다' 라고 메시지가 발생하며 다운로드 실패 UI가 출력됨
	- 하지만, 브라우저 내 영상파일의 실제 다운로드 과정은 진행됨

## 2. 개선 제안 (Improvement)

**[I-01] 사이트 로그 클릭 시 메인 페이지 이동**
- 위치: Header 내 로고 이미지
- 내용: 사이트 로그 클릭 시 메인 페이지로 이동하는 기능이 동작할 때가 있고, 아닐 때가 있음

**[I-02] 사이트 도움말 — 데모 영상 및 인터랙티브 체험 플로우 추가**
- 위치: 메인
- 내용: 메인 페이지 등에서, 서비스 이용 흐름을 이해할 수 있는 데모 영상 또는 인터랙티브 체험 플로우 추가를 제안

**[I-03] 중복 가입 방지 문구 텍스트 개선**
![[회원가입 안내 텍스트 변경 필요.png|447]]
- 위치: 회원가입
- 내용
	- 기타 안내 텍스트는 한글인 반면, 중복 이메일 가입 시도 시 'User already registered' 문구가 발생 
	- 다른 텍스트와의 통일성을 위해 한글로 안내 텍스트 변경

**[I-04] 생성된 TTS 발음의 어색함**
- 위치: 일반제작 > 콘텐츠 검수 (7단계)
- 내용
	- 숫자 발음의 어색함 문제가 종종 발생
		- '1위'를 '은위'로 읽음
		- '200개'를 '유백개'로 읽음
		- '300개'를 '삼박개'로 읽음
		- '5살'을 '오살'로 읽음
		- 숫자는 아니지만, 자동으로 생성된 나레이션 스크립트에 '댓글 ㄱㄱ' 같은 문장도 존재함을 확인. 당연히 이런 문장은 제대로 읽지 못함
	
	- 해당 문제를 우회하기 위한 아래의 방법도 있지만, 활용 과정이 사용자 입장에서 번거롭다는 단점이 있음
		1. 텍스트 및 자막에 표시되는 텍스트를 '숫자'가 아니라 '한글'로 작성 (ex: 300개 -> 삼백개)
		2. 변경된 텍스트를 기반으로, TTS를 재생성 시 '한글'로 작성된 문장을 정상적으로 읽음
		3. 다시 텍스트를 숫자로 변경한 후, 바로 영상을 제작
		4. 해당 과정을 거치면, 생성된 영상의 자막은 숫자로 표시되지만, 한글을 기반으로 TTS가 생성되어 정상적으로 영상이 생성된 것으로 보임

**[I-05] 영상 생성 방식을 선택하지 않고 다음 단계로 넘어갈 수 있는 문제**
![[스크립트 편집 - 영상 생성 방식.png|567]]
- 위치: 일반제작 > 스크립트 편집 (5단계)
- 내용
	- 영상 생성 방식을 선택하지 않고 콘텐츠를 생성한 경우, 오류는 발생하지 않음
	- 하지만, 사용자 입장에서 어떤 기준으로 영상이 생성되었는지 명확하게 파악할 수 없음
		- 바로 다음단계인 콘텐츠 검수에서도 어떤 영상 생성 방식이 적용된 것인지 확인할 수 없음
		- **만약, 환경설정에서 선택한 '기본 영상 생성 방식'이 적용되는 것이라면 UI적으로 선택됨을 명시적으로 보여주는 것이 좋을 듯함**
	
	- 해당 영상 생성 방식의 기본값을 설정하거나, 선택하지 않으면 다음단계로 넘어갈 수 없도록 제한 및 안내 필요

**[I-06] 환경 설정 내 요소 확인**
- 위치: 톱니바퀴 버튼을 눌러 발생하는 환경설정 팝업
- 내용
	- 내가 설정한 기본 설정들이 실제로 잘 적용되고 있는지에 대한 신뢰성이 부족함
		- 배경음 및 음성처럼 실제로 잘 적용되는 요소가 있는 반면, 그렇지 못한 내용 또한 존재함
		
	- 예시: 환경 설정의 제일 상단에서 설정하는 '기본 분야' 가 어느부분에서 적용되는지 알 수 없음
		- 1단계인 분야 선택에서 기본으로 선택되는 것인 줄 알았는데, 아무것도 선택되지 않았으며, 어느 위치에서 동작하는지도 파악할 수 없음. 
		- 즉, 선택한 설정이 어느부분에서 적용되는지에 대한 시인성과 기능이 잘 적용되고 있는지에 대한 신뢰성에 대한 의심 문제가 존재
	
	- 이는, **[I-05]** 와 연관된 문제로 '영상 생성 방식'이 내가 선택한 기본값을 정상적으로 따라갔는지 UI적으로 판단할 수 있도록 해주는 것이 좋다고 생각함
