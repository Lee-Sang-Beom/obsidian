        - Canonical 태그는 검색엔진에 대한 "힌트" 수준에 그쳐 강제성이 없고, 백링크 점수가 도메인별로 분산되는 문제가 발생함

        - `has: [{ type: 'host', value: '...' }]` 조건으로 host 기반 분기 처리, `/:path*` 패턴으로 하위 경로까지 보존하여 리다이렉트
    - 대표 도메인을 `https://www.ijaksnc.com`으로 확정, 나머지 3개 도메인에서 전량 리다이렉트되도록 구성
