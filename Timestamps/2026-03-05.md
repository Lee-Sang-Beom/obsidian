## 1) 설정대상 체크버튼 노출 규칙

* **설정대상 체크버튼이 있는 항목**만 체크 UI 노출
* **없는 항목은 하단 체크박스(체크 UI) 자체를 미노출**
  (비활성/회색 처리 X, **아예 안 보이게**)

## 2) 하단 “승인 / 취소” 버튼 활성화 조건

* 상품 타입이 3개라면,

  * **세 상품 타입 모두에서 “필수 체크”가 완료된 뒤**에만 활성화
* 즉, 활성화 조건은 아래처럼 해석:

  * `상품타입A 완료` AND `상품타입B 완료` AND `상품타입C 완료` → **버튼 활성화**
  * 하나라도 미완료면 → **버튼 비활성화**

## 3) 완료(체크 완료) 판정 기준(권장 정의)

각 상품 타입별로:

* “체크버튼이 존재하는 설정대상”의 **전체 개수 = N**
* 그 중 사용자가 체크한 개수 = C
* **C == N 일 때 해당 상품 타입은 완료**
