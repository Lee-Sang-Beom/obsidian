
####  Work (프로젝트: 공통 AI 검증코어)
---
- 4-Layer Architecture 문서 기반으로 Layer 2(Knowledge), Layer 3(RAG Harness) 구현 착수
    - `loader.py` : PDF 문서 로드 및 문장 단위 청킹 구현
    - `embedder.py` : BAAI/bge-m3 임베딩 모델 적용, 벡터 변환 및 `.pt` 파일 저장/로드 구현
    - `retriever.py` : 벡터 검색(Cosine Similarity) + BM25 키워드 검색 결합한 하이브리드 검색 구현
    - `reranker.py` : BAAI/bge-reranker-v2-m3 적용, Cross-Encoder 기반 리랭킹 구현
    - `main.py` : 전체 파이프라인 통합 실행 진입점 구현

- 실제 PDF 문서(RAG 기반 AI 환각 완화 시스템 구축을 위한 오픈소스 생태계 조사)를 타겟 문서로 설정, 파이프라인 전체 흐름 동작 확인
