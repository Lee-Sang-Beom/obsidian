	- `<StatisticsFilterContainer/>` : 필터 이벤트 핸들링을 담당하는 컴포넌트로 분리됨. 
		- `handleDateRangeChange`로 날짜 범위 변경 시 첫 페이지(nowPage: 1)로 리셋
		- `handleSearch`로 검색 버튼 클릭 시 첫 페이지로 이동
		- `handleReset`으로 모든 필터 초기화(dateRange: undefined, nowPage: 1, pageSize: 10)
		- 상태는 부모로부터 `filters` prop으로 받고 변경사항은 `onFiltersChange` 콜백으로 전달
	- `<StatisticsFilterPresenter/>` : Container-Presenter 패턴 적용으로 UI 렌더링만 담당할 수 있도록 함.
		- `SearchFilter` 컴포넌트에 전달할 `searchFields` 배열 구성 (dateRange 타입 필터 1개). 
		- 모든 상태(dateRange)와 이벤트 핸들러(onDateRangeChange, onSearch, onReset)를 props로 받아 `SearchFilter`에 전달. 
		- 리팩토링된 `SearchFilter` 컴포넌트를 활용할 수 있도록 변경
