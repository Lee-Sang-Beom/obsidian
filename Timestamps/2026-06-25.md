
####  Work (프로젝트: 공통 AI 검증코어)
---
- 로컬 스웨거 환경의 API 요소에 대한 필드 설명 추가
- LLM 답변을 법령 근거 검증 가능한 작은 주장들로 분해하는 준비단계진행
- LLM 답변 내 주장을 검증할 때 봐야 할 법령 근거를 연결하는 작업진행 
- 연결된 근거가 실제로 주장을 뒷받침하는지에 대한 1차판정작업진행
- 검증 결과를 실행 가능한 판단으로 바꾸는 단계 진행
	- 검증 결과에 따라 다음 행동을 결정할 수 있는 상태를 판단
```
모든 claim이 supported
  -> pass
근거가 부족한 claim이 있음
  -> retrieve_more
근거는 있지만 아직 판단이 안 된 claim이 있음
  -> revise
source integrity 같은 구조적 high severity 문제가 있음
  -> human_review
claim이 근거와 반대됨
  -> block
  
GateDecision
pass / retrieve_more / revise / human_review / block 중 하나로 결정
```
