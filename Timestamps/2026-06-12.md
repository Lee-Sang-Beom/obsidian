- 2주차 회의 결과로, 당장 이번주에 해볼것은 아래와 같음
	- 법령 API 정보를 크로마에 저장
	- 검색엔진 및 병렬로 질문 임베딩/하이브리드 연결진행

- RAG 파이프라인 구현 코드를 Step 1 레퍼런스 아키텍처 기준으로 리팩터링 진행
	- `loader.py` : 청킹 방식을 문장 분리 regex에서 max_chars 슬라이딩 분할로 교체, 한국어 법령 본문 오분할 방지
	- `embedder.py` : `normalize_embeddings=True` 기본 적용, `as_float_matrix` 헬퍼 추가(외부 인덱스 호환)
	- `retriever.py` : 반환 타입을 `list[tuple[str, float]]`로 통일, 코사인 유사도 계산 단순화
	- `reranker.py` : `DeterministicReranker` 폴백 도입, BGE 모델 없는 환경에서도 파이프라인 동작 보장
	- `law_fetcher.py` : `requests` 의존 제거 후 `urllib` 기반으로 교체, `AdapterError`로 예외 일관화
	- `main.py` : BGE reranker 로드 실패 시 자동 폴백 처리, `as_float_matrix` 재사용으로 중복 제거
