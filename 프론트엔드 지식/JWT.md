#### 1. JWT 토큰이란?

- JWT(Json Web Token)는 JSON 객체에 인증에 필요한 정보들을 담은 후 비밀키로 서명한 토큰이다.
- **JSON** 포맷을 이용하여 사용자에 대한 속성을 저장하는 **Claim** 기반의 Web 토큰으로, JWT는 필요한 정보를 자체적으로 지니는 **Self-Contained** 방식으로 정보를 안정성 있게 전달한다.
